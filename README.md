# Chevruta.AI

An AI-powered Jewish source sheet builder.

**Chevruta.AI** helps you build beautiful Jewish source sheets in seconds. It features:
*   **Intelligent Suggestions**: Powered by Google's Gemini API to suggest relevant sources.
*   **Sefaria Integration**: Direct access to the Sefaria library for texts.
*   **Instant Export**: Export to PDF or Google Docs.
*   **Modern Interface**: A clean, distraction-free environment for Torah study.

## Development

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Start the dev server: `npm run dev`
